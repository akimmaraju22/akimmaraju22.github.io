Projects

Project 1: Queens Problem

Designed using Java and Processing as a GUI. This project visualizes the queens problem and implements the algorithm to keep all the Queens in a position where they cant attack each other. Application allows you to control the size of the chessboard using the '+' and '-' and number keys '4' to '9' NOTE: Application needs Java version 17 installed on your device to run.

Project 2: Bubble Sort Visualization

This project is programmed using Java and Processing as a GUI. The program visualizes the bubble sort algorithm step by step using the buttons in the application that sort through an array of 25 elements. The application also detects if the array is sorted and outputs the array as a histogram that represents each element by size with respect to other values in the array. NOTE: Application needs Java version 17 installed on your device to run.

Download Project 2
